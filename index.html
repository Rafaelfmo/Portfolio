Sobre Mim

Meu nome é Rafael Fonseca Mota de Oliveira, sou ex-aluno de turma ITA e Olimpíadas, além de ter sido aprovado na UFC. Atualmente, sou estudante de Ciência da Computação na Universidade de Fortaleza e estou em busca de oportunidades como Desenvolvedor de Software com meus conhecimentos nas diversas bibliotecas e frameworks das linguagens Python e JavaScript.

Em Python, estou familiarizado com bibliotecas poderosas como Pandas, Numpy, BeautifulSoup e Selenium para diversos tipos de projetos. Flask e Django para desenvolvimento web em diferentes tipos de Bancos de dados, sejam eles SQL ou NoSQL.

Em Javascript, conheço não apenas jQuery e React.js para criar interfaces de usuário dinâmicas, mas também Node.js para o desenvolvimento de aplicativos no lado do servidor, combinado com Express.js para construir APIs RESTful. Além disso, gosto de utilizar a implementação de padrões de projeto durante o desenvolvimento.
